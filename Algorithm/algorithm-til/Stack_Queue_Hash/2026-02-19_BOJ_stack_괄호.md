## 🧩 문제 설명

여러 줄의 괄호 문자열이 주어질 때, 각 문자열이 VPS(올바른 괄호 문자열)인지 판별하는 문제다.  
올바르면 `YES`, 아니면 `NO`를 출력한다.

---

## 🧩 문제 핵심 포인트

* `'('`를 만나면 스택에 넣고, `')'`를 만나면 스택에서 꺼내 매칭한다.
* `')'`를 만났는데 스택이 비어 있으면 즉시 `NO`다.
* 문자열을 다 확인한 뒤 스택이 비어 있어야 `YES`다.

---

## 🧩 내 풀이 방식

* 입력 문자열을 한 글자씩 순회한다.
* `'('`는 `stack.push()` 한다.
* `')'`는 스택이 비어 있으면 바로 `"NO"` 반환, 아니면 `stack.pop()` 한다.
* 순회가 끝난 뒤 스택 길이가 `0`이면 `"YES"`, 아니면 `"NO"`를 반환한다.
* 테스트케이스 수 `n`만큼 반복해서 결과를 줄바꿈으로 출력한다.

---

## 🧩 코드 로직 정리

### 전체 흐름

```txt
1. 테스트케이스 개수 n 입력
2. 각 문자열에 대해 solution 실행
3. 문자를 순회하며 '('는 push, ')'는 pop 시도
4. ')'에서 pop 불가(스택 비어있음)면 NO 반환
5. 순회 종료 후 stack이 비어있으면 YES, 아니면 NO
6. 모든 결과를 줄바꿈 출력
```

---

## 🧩 코드 구현 (내 풀이)

```js
const fs = require("fs");
const input = fs.readFileSync(0, "utf-8").trim().split("\n");

function solution(arr) {
  const arrSplit = [...arr];
  const stack = [];

  for (const e of arrSplit) {
    if (e === "(") {
      stack.push(e);
    }

    if (e === ")") {
      if (stack.length === 0) return "NO";
      stack.pop();
    }
  }

  return stack.length === 0 ? "YES" : "NO";
}

const n = Number(input[0]);
const answer = [];

for (let i = 1; i <= n; i++) {
  answer.push(solution(input[i]));
}

console.log(answer.join("\n"));
```

## 🧩 사용된 JS 개념

* 전개 연산자 `...` : 문자열을 문자 배열로 변환
* 배열 `push()` / `pop()` : 스택 동작 구현
* `for...of` : 문자열 문자 단위 순회

## ⚠️ 오답 포인트

* `')'`를 만났을 때 스택 빈 상태를 먼저 체크하지 않으면 런타임 오류/오답이 난다.
* 순회 후 스택 잔여 원소(`'('`)를 확인하지 않으면 잘못 `YES`를 출력할 수 있다.
* 테스트케이스별로 스택을 초기화하지 않으면 이전 결과가 섞인다.
