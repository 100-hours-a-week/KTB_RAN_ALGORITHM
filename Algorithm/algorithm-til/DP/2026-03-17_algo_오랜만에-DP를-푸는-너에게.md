# 오랜만에 DP를 푸는 너에게

### 나만의 DP 풀기

전에 RGB 문제 풀 때 이런 생각의 흐름으로 풀었었다.

1. DP 정의하기: **i번째 특정 색상일 때 최소 비용**
2. i번째에 해당 상황에서 내가 할 수 있는 선택
   1. DP는 다른 것을 생각하지 않는다. 항상 현재 상황을 생각한다.
      1. **i번째 집을 빨강으로 칠했을 때 최소 비용 (헷갈리지 않게 문장을 모두 구사하기)**
3. 현재를 정의한 후에, 이전의 상태를 작성한다.
   1. 이때 최솟값을 구하는 것이라면 바로 직전 상태 중 최소 고르기
   2. **항상 이전의 상태는 바로 직전 상태만 생각하기.**
      1. **DP는 지금에 충실한다. 멀리 가지 않는다. 마치 우리 인생처럼...**

인사이트: 최솟값을 구할 때는 **(과거 중 최솟값) + 현재(필수)** 조합이다.
