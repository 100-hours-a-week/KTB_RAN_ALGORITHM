- **문제 레벨** : Gold 4

- **문제 링크** : https://www.acmicpc.net/problem/11657

## 🧩 문제 설명

도시 `N`개와 버스 노선 `M`개가 주어진다.  
1번 도시에서 출발해 다른 도시들로 가는 최소 시간을 구해야 한다.

- 도달할 수 없는 도시는 `-1`
- 1번 도시에서 도달 가능한 음수 사이클이 있으면 결과 전체를 `-1` 한 줄만 출력

---

## 🧩 문제 핵심 포인트

### 벨만-포드 특징

- 음수 가중치가 있어도 단일 시작점 최단거리를 구할 수 있다.
- 모든 간선을 `N-1`번 완화하면 최단거리가 확정된다.
- `N`번째 완화에서 갱신이 발생하면, 시작점에서 도달 가능한 음수 사이클이 존재한다.

### 구현 팁

- 간선은 인접 리스트보다 `edges = [(u, v, w), ...]` 형태가 벨만-포드 반복에 직관적이다.
- 완화할 때 `dist[u] != INF` 조건을 꼭 넣어야 "도달 불가능 정점"에서 잘못 전파되지 않는다.
- 한 라운드에서 갱신이 한 번도 없으면 조기 종료(`break`)해도 정답은 같다.

---

## 🧩 내 풀이 방식

- `dist[1] = 0`, 나머지는 `INF`로 초기화했다.
- 간선 리스트를 `N-1`번 순회하며 완화(relaxation)를 반복했다.
- 마지막으로 간선을 한 번 더 순회해 갱신 가능 여부를 확인했고, 가능하면 음수 사이클로 판단했다.
- 음수 사이클이 없으면 `2..N`의 최단거리를 출력하고, `INF`인 정점은 `-1`로 출력했다.

---

## 🧩 코드 로직 정리

### 전체 흐름

```txt
1. N, M 입력 후 간선 리스트 edges 구성
2. dist 배열을 INF로 초기화, dist[1] = 0
3. N-1번 반복:
     모든 간선 (u, v, w)에 대해
     dist[u]가 유효하고 dist[v] > dist[u] + w 이면 갱신
4. 모든 간선을 1번 더 확인:
     갱신 가능하면 음수 사이클 -> -1 출력 후 종료
5. i=2..N:
     dist[i]가 INF면 -1, 아니면 dist[i] 출력
```

### 핵심 조건

1. `if dist[u] != INF`
   - 시작점에서 닿을 수 있는 경로만 완화해야 한다.
2. 음수 사이클 판별은 `N-1`회 이후 "추가 1회 검사"로 한다.
3. 문제 요구상 음수 사이클이 있으면 도시별 결과 대신 `-1` 한 줄만 출력한다.

## 🧩 코드 구현 (내 풀이)

```python
import sys

sys.stdin = open("../예제.txt")
input = sys.stdin.readline

N, M = map(int, input().split())

edges = []
for _ in range(M):
    a, b, c = map(int, input().split())
    edges.append((a, b, c))

INF = float("inf")
dist = [INF] * (N + 1)
dist[1] = 0

# V-1번 완화
for _ in range(N - 1):
    updated = False
    for u, v, w in edges:
        if dist[u] != INF and dist[v] > dist[u] + w:
            dist[v] = dist[u] + w
            updated = True
    if not updated:
        break

# 음수 사이클 체크
for u, v, w in edges:
    if dist[u] != INF and dist[v] > dist[u] + w:
        print(-1)
        exit()

# 출력
for i in range(2, N + 1):
    if dist[i] == INF:
        print(-1)
    else:
        print(dist[i])
```

---

## 🧩 사용된 Python 개념

- `list`로 간선 리스트 구성: `edges.append((u, v, w))`
- `float("inf")`로 초기 무한대 거리 표현
- 반복 완화(relaxation) 패턴
- `updated` 플래그로 조기 종료 최적화

## ⚠️ 오답 포인트

- `dist[u] != INF` 없이 완화하면 도달 불가능 정점에서도 거리 갱신이 일어나 오답이 난다.
- 음수 사이클 판별을 `N-1`회 안에서 처리하려 하면 조건이 불안정해진다.
- 음수 사이클이 있는 경우 도시별 값을 출력하면 문제 출력 형식과 다르다.
- 도달 불가능 정점을 `0`으로 출력하면 오답이다. 이 문제는 `-1`이다.
