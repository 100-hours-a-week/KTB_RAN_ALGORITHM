## 🧩 문제 설명

`n x m` 게임 맵에서 `(0, 0)`에서 `(n-1, m-1)`까지 이동할 때,  
지나갈 수 있는 칸(`1`)만 사용해서 도착점까지 가는 최단거리를 구하는 문제이다.  
도달할 수 없으면 `-1`을 반환한다.

---

## 🧩 문제 핵심 포인트

- 간선 가중치가 모두 동일한 격자 최단거리 문제라 BFS가 정석이다.
- `dist[next] = dist[cur] + 1`로 최단거리 누적이 된다.
- 방문하지 않은 칸 + 이동 가능한 칸(`maps[nextX][nextY] == 1`)만 큐에 넣는다.

---

## 🧩 내 풀이 방식

- `visited`, `dist` 2차원 배열을 만든다.
- 시작점 `(0,0)`을 방문 처리하고 `dist[0][0] = 1`로 시작한다.
- 큐에서 현재 좌표를 꺼내 4방향을 확인한다.
- 범위 내 + 미방문 + 이동 가능 칸이면 방문 처리 후 거리 갱신하고 큐에 추가한다.
- BFS 종료 후 `dist[n-1][m-1]`가 0이면 도달 불가이므로 `-1`을 반환한다.

---

## 🧩 코드 로직 정리

### 전체 흐름

```txt
1. n, m 크기 계산
2. visited, dist 2차원 배열 초기화
3. 시작점 (0,0) 큐 삽입 및 dist=1 설정
4. BFS 수행: 4방향 이웃 검사
5. 유효한 이웃이면 방문 처리 + dist 갱신 + 큐 삽입
6. BFS 종료 후 도착점 dist 값 확인
7. 값이 있으면 거리 반환, 없으면 -1 반환
```

---

## 🧩 코드 구현 (내 풀이)

```python
from collections import deque

def solution(maps):
    n = len(maps)
    m = len(maps[0])
    visited = [ [False]*m for _ in range(n)]
    dist = [ [0]*m for _ in range(n)]

    start = [0,0]
    visited[0][0] = True
    dist[0][0] = 1
    queue = deque([start])


    dx = [0,1,-1,0]
    dy = [1,0,0,-1]

    while queue :
        curX, curY = queue.popleft()

        for i in range(4) :
            nextX = curX + dx[i]
            nextY = curY + dy[i]

            if (0 <= nextX < n and 0 <= nextY < m) and not visited[nextX][nextY] and maps[nextX][nextY] == 1:
                visited[nextX][nextY] = True
                dist[nextX][nextY] = dist[curX][curY] + 1
                queue.append([nextX,nextY])


    if dist[n-1][m-1] :
        return dist[n-1][m-1]
    else  :
        return -1
```

---

## 🧩 사용된 Python 개념

- `collections.deque` : BFS 큐의 `popleft()`를 `O(1)`로 처리
- 2차원 리스트 초기화 : `[[...]*m for _ in range(n)]`
- 경계 검사 : `0 <= nx < n and 0 <= ny < m`
- 조건 결합 : 방문 여부 + 이동 가능 칸을 한 번에 검사

---

## 🧩 복잡도

- 시간복잡도 : `O(n*m)` (각 칸 최대 1번 방문)
- 공간복잡도 : `O(n*m)` (`visited`, `dist`, `queue`)

## ⚠️ 오답 포인트

- `dist[0][0]`을 `0`으로 두면 칸 수 기준 문제에서 오프바이원 실수가 난다.
- 방문 처리를 큐에서 꺼낼 때 하면 같은 칸이 여러 번 큐에 들어갈 수 있다.
- `maps[nextX][nextY] == 1` 조건을 빼먹으면 벽 칸으로 이동해 오답이 난다.
