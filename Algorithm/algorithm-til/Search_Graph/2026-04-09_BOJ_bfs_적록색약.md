---

## 🧩 문제 설명

`N x N` 격자에서 색 영역 개수를 구하는 문제이다.  
일반 시야 기준 영역 수와, 적록색약(빨강/초록을 같은 색으로 보는 경우) 기준 영역 수를 각각 출력해야 한다.

---

## 🧩 문제 핵심 포인트

- 격자에서 인접한 같은 색을 하나의 영역으로 묶는 전형적인 BFS 문제다.
- 일반 시야와 적록색약 시야를 각각 탐색해야 하므로 그래프 상태를 분리해야 한다.
- 색약 케이스는 `G -> R`로 그래프를 변형해두면 BFS 조건이 단순해진다.

---

## 🧩 내 풀이 방식

- 입력 그래프를 받고, `defGraph = [row[:] for row in graph]`로 깊은 복사한다.
- `defGraph`에서 `G`를 `R`로 치환해 색약용 그래프를 만든다.
- 같은 BFS 함수를 두 번 사용한다.
- 첫 번째는 원본 `graph`로 일반 영역 수(`count`)를 세고,
- 두 번째는 `defGraph`로 색약 영역 수(`defCount`)를 센다.

---

## 🧩 코드 로직 정리

### 전체 흐름

```txt
1. graph 입력
2. defGraph를 깊은 복사
3. defGraph의 G를 R로 치환
4. BFS로 graph 영역 개수 count 계산
5. visited 초기화
6. BFS로 defGraph 영역 개수 defCount 계산
7. count, defCount 출력
```

---

## 🧩 코드 구현 (내 풀이)

```python
import sys
from collections import deque


sys.stdin = open("../예제.txt")
input = sys.stdin.readline
n = int(input())

graph = [list(input().strip()) for _ in range(n)]
defGraph = [row[:] for row in graph]
for i in range(n) :
    for j in range(n) :
        if defGraph[i][j] == "G" :
            defGraph[i][j] = "R"



#방문노드
visited = [[False]*n for _ in range(n)];

#옆으로 퍼지는 거면 가는 방향을 4방향으로 해야됨.
dx = [1,0,0,-1]
dy = [0,1,-1,0]

count = 0;
defCount = 0;

#BFS로 진행하는데, 적록색약은 초록빨강을 하나로 생각.
#적록색약일 경우도 해야됨.
#적록색약은 빨강초록의 경우를 합하면됨.

def BFS(start,graph) :
    #발자취 큐
    x,y=start
    queue = deque();
    visited[x][y] = True
    #BFS
    queue.append(start)
    color = graph[x][y]


    while queue :
        curX,curY = queue.popleft()
        #시작한 색상을 시준으로 시작

        #시작한 색상이 빨간색일 경우 빨강 count에 들어감
        #초록일경우 초록 count에 들어감

        for i in range(4) :
            nx = curX + dx[i]
            ny = curY + dy[i]

            if 0 <= nx < n and 0<= ny <n :
                if graph[nx][ny] == color and not visited[nx][ny] :
                    visited[nx][ny] = True
                    queue.append([nx,ny])




#그래프 시작이 방문하지 않았을 경우, BFS를 시작한다.
for i in range(n) :
    for j in range(n) :
        if not visited[i][j] :
            color = BFS([i,j],graph)
            count+=1

visited = [[False]*n for _ in range(n)];

for i in range(n) :
    for j in range(n) :
        if not visited[i][j] :
            color = BFS([i,j],defGraph)
            defCount+=1
#적록색일 경우,
print(count, defCount)
```

## 🧩 사용된 Python 개념

- `deque` : BFS 큐 구현
- 2차원 리스트 순회 : 모든 시작점 검사
- 2차원 리스트 깊은 복사 : `defGraph = [row[:] for row in graph]`
- 그래프 전처리(값 치환) : `G -> R`

## ⚠️ 오답 포인트

- `defGraph = graph`처럼 얕은 복사를 하면 원본도 같이 바뀐다.
- 방문 배열을 재사용하면서 초기화하지 않으면 두 번째 탐색 결과가 틀린다.
- BFS 조건에 색약 예외를 직접 많이 넣으면 분기가 복잡해지고 실수가 늘어난다.

## 💡 구현 팁 (이번 인사이트 요약)

- 2차원 깊은 복사는 이 패턴을 기억하면 된다.

```python
defGraph = [row[:] for row in graph]
```

- 조건문을 복잡하게 늘리기보다, 그래프를 전처리해서 문제를 단순화할 수 있다.
- `R/G`를 하나로 취급해야 하면 BFS 내부 분기보다 그래프 치환(`G -> R`)이 유지보수에 유리하다.
- 오랜만에 격자 BFS를 할 때는 시작 루틴을 고정하면 덜 헷갈린다.
  - `for i, j` 전체 순회
  - `if not visited[i][j]: BFS([i, j])`
  - 영역 카운트 `+1`
