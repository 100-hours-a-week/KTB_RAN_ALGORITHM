## 🧩 문제 설명

`N`개의 도시와 `M`개의 단방향 도로가 주어진다.  
출발 도시 `X`에서 최단거리가 정확히 `K`인 모든 도시 번호를 오름차순으로 출력하는 문제이다.  
없으면 `-1`을 출력한다.

---

## 🧩 문제 핵심 포인트

- 간선 가중치가 모두 `1`이라서 다익스트라보다 BFS가 직관적이다.
- `dist` 배열 하나로 방문 여부와 거리 정보를 동시에 관리할 수 있다.
- 정답은 `dist[i] == K`인 도시만 모으면 된다.

---

## 🧩 BFS 핵심 공식 4가지 (복기)

로직은 어렵지 않은데, 이 4가지를 놓치면 BFS가 자주 꼬인다.

```text
1) queue에 시작점 넣기
2) dist[start] = 0 (또는 visited[start] = true)
3) 꺼낸 current의 이웃 next 순회
4) 미방문이면 dist[next] = dist[current] + 1 후 queue에 push
```

`dist`를 쓰는 패턴에서는 보통 `-1`을 미방문으로 둔다.

---

## 🧩 내 풀이 방식

- 인접 리스트 `graph`를 만든다. (`graph[u].append(v)`)
- `dist = [-1] * (n+1)`로 초기화하고 시작점 `x`는 `0`으로 둔다.
- BFS를 한 번 돌면서 도달 가능한 모든 도시의 최단거리를 채운다.
- 마지막에 `dist[i] == k`인 도시만 출력 배열에 담아 출력한다.

---

## 🧩 코드 로직 정리

### 전체 흐름

```txt
1. n, m, k, x 입력
2. graph(인접 리스트) 생성
3. dist 배열을 -1로 초기화
4. BFS 시작: dist[x] = 0, queue = [x]
5. 이웃 도시를 보면서 dist[next]가 -1이면 거리 갱신 후 큐에 삽입
6. BFS 종료 후 dist[i] == k 인 도시를 모아 출력
7. 없으면 -1 출력
```

---

## 🧩 코드 구현 (내 풀이)

```python
import sys
from collections import deque

sys.stdin = open("../예제.txt")
input = sys.stdin.readline

#알고리즘 아이디어
#BFS로 이웃들을 모두 큐를 넣어서 반복하여 돌릴때, 받은 input()값으로 큐에서 뺀다.
#조건이 2의 거리일 경우, 2번 돌려서 있을 경우 -> 배출 없을 경우 -1

#파이썬으로 BFS 어캐함..?


n,m,k,x = map(int,input().split())
#n : 도시 노드 갯수
#m : 길 갯수
#k : 원하는 거리
#x : 시작 노드

graph = [ [] for _ in range(n+1)]
dist = [-1]*(n+1)

for i in range(m) :
    u,v = map(int,input().split())
    graph[u].append(v)


#1,(depth = 0) / 2,3 (depth = 1) / 4 (depth = 2)
#bfs 할 때 1. 시작 노드, 2. 큐(탐색노드) 3.visited
def bfs() :
    queue = deque()
    queue.append(x);
    dist[x] = 0

    while queue :
        curX = queue.popleft();

        for next in graph[curX] :
            if dist[next] == -1 :
                dist[next] = dist[curX] + 1
                queue.append(next)

    return dist


answer = bfs()
answerArr = [];

for i in range(1, n+1) :
    if answer[i] == k :
        answerArr.append(i);

if answerArr :
    print("\n".join(map(str,answerArr)))
else :
    print("-1")
```

---

## 🧩 사용된 Python 개념

- `collections.deque` : BFS 큐의 `popleft()`를 `O(1)`로 처리
- 인접 리스트 : `graph = [[] for _ in range(n+1)]`
- 거리 배열 초기화 : `dist = [-1] * (n+1)` (미방문 표기)
- `"\n".join(map(str, arr))` : 다중 줄 출력 포맷

---

- 시간복잡도 : `O(N + M)`
- 공간복잡도 : `O(N + M)`

## ⚠️ 오답 포인트

- 방문 처리를 늦게 하거나 빼먹으면 중복 방문으로 시간 낭비가 커진다.
- `dist` 초기값을 `0`으로 두면 시작점과 미방문을 구분하기 어렵다.
- 정답 도시를 모을 때 `dist[i] <= k`로 잘못 비교하면 오답이 된다.

- 다음 BFS 문제부터는 시작 전에 아래 4줄을 먼저 적고 시작한다.

  - `dist[start] = 0`
  - `for next in graph[cur]:`
