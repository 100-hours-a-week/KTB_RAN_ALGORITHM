## 🧩 문제

- **문제 제목** : 나이트의 이동
- **문제 레벨** : Silver I
- **문제 유형** : BFS, 최단거리
- **문제 제공** : BOJ
- **문제 링크** : https://www.acmicpc.net/problem/7562
- **코드 파일** : [knight-move-7562.js](../../algorithm-JS/Search_Graph/knight-move-7562.js)

---

## 🧩 문제 설명

체스판 크기 `l x l`이 주어진다.  
나이트는 한 번에 8가지 방향으로만 이동할 수 있다.

각 테스트케이스마다 시작 칸에서 목표 칸까지 이동할 때,  
최소 몇 번의 이동이 필요한지 출력하는 문제다.

---

## 🧩 문제 핵심 포인트

- 나이트 이동 규칙은 고정 8방향이므로 방향 벡터로 관리하면 된다.
- "최소 이동 횟수" 문제이므로 BFS를 쓰면 최초 도달 거리가 곧 정답이다.
- 테스트케이스마다 `visited`, `dist`, `queue`를 새로 초기화해야 값이 섞이지 않는다.

---

## 🧩 내 풀이 방식

- 시작점이 목표점과 같으면 `0`을 바로 반환했다.
- `visited`로 재방문을 막고, `dist`로 현재까지 이동 횟수를 저장했다.
- BFS 중 이웃 칸을 방문할 때 `dist[next] = dist[cur] + 1`로 누적했다.
- 목표 칸을 발견하는 즉시 해당 거리를 반환해 불필요한 탐색을 줄였다.

---

## 🧩 코드 로직 정리

### 전체 흐름

```txt
1. 테스트케이스 수 t를 읽는다.
2. 각 테스트케이스마다 체스판 크기, 시작 좌표, 목표 좌표를 읽는다.
3. 시작 좌표를 큐에 넣고 BFS를 시작한다.
4. 현재 좌표에서 나이트 8방향으로 다음 좌표를 계산한다.
5. 범위 안이고 미방문이면 방문 처리 + 거리 갱신 후 큐에 넣는다.
6. 목표 좌표를 만나면 즉시 거리 값을 반환한다.
```

## 🧩 코드 구현 (내 풀이)

### JavaScript

```js
const fs = require("fs"); // /dev/stdin
const input = fs.readFileSync("../예제.txt").toString().trim().split("\n");

//백준 제출 시
// const fs = require("fs");
// const input = fs.readFileSync(0, "utf-8").trim().split("\n");

const t = Number(input[0]);
let line = 1;
const answer = [];

for (let tc = 0; tc < t; tc++) {
  const n = Number(input[line++]);
  const [startX, startY] = input[line++].split(" ").map(Number);
  const [endX, endY] = input[line++].split(" ").map(Number);

  answer.push(solution(n, startX, startY, endX, endY));
}

console.log(answer.join("\n"));

function solution(n, startX, startY, endX, endY) {
  if (startX === endX && startY === endY) return 0;

  const visited = Array.from({ length: n }, () => Array(n).fill(false));
  const dist = Array.from({ length: n }, () => Array(n).fill(0));
  const queue = [];
  let head = 0;

  const dx = [-2, -2, -1, 1, 2, 2, -1, 1];
  const dy = [-1, 1, 2, 2, 1, -1, -2, -2];

  queue.push([startX, startY]);
  visited[startX][startY] = true;

  while (head < queue.length) {
    const [curX, curY] = queue[head++];

    for (let i = 0; i < 8; i++) {
      const nextX = curX + dx[i];
      const nextY = curY + dy[i];

      if (
        nextX >= 0 &&
        nextX < n &&
        nextY >= 0 &&
        nextY < n &&
        !visited[nextX][nextY]
      ) {
        visited[nextX][nextY] = true;
        dist[nextX][nextY] = dist[curX][curY] + 1;

        if (nextX === endX && nextY === endY) {
          return dist[nextX][nextY];
        }

        queue.push([nextX, nextY]);
      }
    }
  }

  return dist[endX][endY];
}
```

### Python

```python
import sys
from collections import deque

sys.stdin = open("../예제.txt")
input = sys.stdin.readline

t = int(input())


def solution(n, startX, startY, endX, endY):
    queue = deque()
    visited = [[False] * n for _ in range(n)]
    dist = [[0] * n for _ in range(n)]

    dx = [-2, -2, -1, 1, 2, 2, -1, 1]
    dy = [-1, 1, 2, 2, 1, -1, -2, -2]

    queue.append((startX, startY))
    visited[startX][startY] = True

    while len(queue) != 0:
        curX, curY = queue.popleft()

        for i in range(8):
            nextX = curX + dx[i]
            nextY = curY + dy[i]

            if 0 <= nextX < n and 0 <= nextY < n and not visited[nextX][nextY]:
                queue.append((nextX, nextY))
                visited[nextX][nextY] = True
                dist[nextX][nextY] = dist[curX][curY] + 1

    return dist[endX][endY]


for _ in range(t):
    n = int(input())
    x1, y1 = map(int, input().split(" "))
    x2, y2 = map(int, input().split(" "))

    print(solution(n, x1, y1, x2, y2))
```

## 🧩 사용된 JS 개념

- `Array.from({ length: n }, () => Array(n).fill(...))` : 2차원 배열 초기화
- `queue` + `head` 인덱스 : `shift()` 없이 BFS 큐 순회
- 구조분해 할당 : `[x, y]` 좌표를 읽고 관리

---

## 🧩 복잡도

- 시간복잡도 : `O(l^2)` (각 칸을 최대 1번 방문)
- 공간복잡도 : `O(l^2)` (`visited`, `dist`, `queue`)

## ⚠️ 오답 포인트

- 테스트케이스마다 배열을 재사용하면 이전 케이스 값이 남아 오답이 난다.
- `queue.shift()`를 반복하면 입력이 커질 때 성능이 급격히 떨어질 수 있다.
- 시작점과 도착점이 같은 경우를 따로 처리하지 않으면 불필요한 BFS를 수행한다.
