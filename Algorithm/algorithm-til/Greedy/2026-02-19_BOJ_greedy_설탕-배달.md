## 🧩 문제 설명

`3kg` 봉지와 `5kg` 봉지를 이용해 정확히 `Nkg`을 만들어야 한다.  
봉지 개수를 최소로 하되, 정확히 만들 수 없으면 `-1`을 출력한다.

---

## 🧩 문제 핵심 포인트

* 봉지 개수를 최소로 하려면 큰 단위(`5kg`)를 최대한 먼저 써야 한다.
* `5kg` 개수를 줄여 가며 나머지가 `3kg`로 나누어떨어지는지 확인하면 된다.
* 가능한 조합을 찾는 즉시 그 값이 최소 봉지 개수다.

---

## 🧩 내 풀이 방식

* `5kg` 봉지 개수를 `Math.floor(n / 5)`로 시작한다.
* 현재 `5kg` 개수에서 남는 무게(`remain`)를 계산한다.
* `remain % 3 === 0`이면 `five + remain / 3`을 반환한다.
* 아니면 `5kg` 개수를 1개 줄이고 반복한다.
* 끝까지 못 찾으면 `-1`을 반환한다.

---

## 🧩 코드 로직 정리

### 전체 흐름

```txt
1. five = floor(n / 5)
2. five가 0 이상인 동안 반복
3. remain = n - five * 5
4. remain이 3으로 나누어떨어지면 five + remain/3 반환
5. 아니면 five를 1 감소
6. 반복 종료까지 못 찾으면 -1 반환
```

---

## 🧩 코드 구현 (내 풀이)

```js
const fs = require("fs");
const input = fs.readFileSync(0, "utf-8").trim();

function solution(n) {
  let five = Math.floor(n / 5);

  while (five >= 0) {
    const remain = n - five * 5;
    if (remain % 3 === 0) return five + remain / 3;
    five -= 1;
  }

  return -1;
}

const n = Number(input);
console.log(solution(n));
```

---

## 🧩 사용된 JS 개념

* `Math.floor()` : `5kg` 봉지를 최대로 담을 수 있는 개수 계산
* `%` (나머지 연산) : 남은 무게가 `3kg`로 딱 나누어지는지 확인
* `while` 반복문 : `5kg` 봉지 개수를 줄여가며 탐색
