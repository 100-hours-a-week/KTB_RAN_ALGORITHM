## 🧩 문제 설명

회원 `N`명의 나이와 이름이 가입 순서대로 주어진다.  
나이가 증가하는 순으로 정렬하되, 나이가 같으면 가입한 순서를 유지해서 출력해야 한다.

---

## 🧩 문제 핵심 포인트

* 1차 정렬 기준은 나이 오름차순이다.
* 나이가 같은 경우 입력 순서를 유지해야 하므로 안정 정렬 개념이 필요하다.
* 입력을 객체 배열로 변환한 뒤 정렬하고 출력 문자열로 변환하면 된다.

---

## 🧩 내 풀이 방식

* 입력 문자열을 순회하며 `{ age, name }` 객체를 만든다.
* `members.sort((a, b) => a.age - b.age)`로 나이 기준 정렬한다.
* 정렬 결과를 `"age name"` 형태로 매핑해 줄바꿈으로 출력한다.

---

## 🧩 코드 로직 정리

### 전체 흐름

```txt
1. n, 회원 정보 배열 입력
2. 각 줄을 age, name으로 분리해 객체 배열 생성
3. age 기준 오름차순 정렬
4. "age name" 형식으로 변환
5. 줄바꿈(join) 후 출력
```

---

## 🧩 코드 구현 (내 풀이)

```js
const fs = require("fs");
const input = fs.readFileSync(0, "utf-8").trim().split("\n");

function solution(n, arr) {
  const members = [];

  for (const e of arr) {
    const [age, name] = e.split(" ");
    members.push({ age: Number(age), name });
  }

  members.sort((a, b) => a.age - b.age);

  return members.map((m) => `${m.age} ${m.name}`).join("\n");
}

const n = Number(input[0]);
const arr = input.slice(1, n + 1);

console.log(solution(n, arr));
```

---

## 🧩 사용된 JS 개념

* `split(" ")` : 나이와 이름 파싱
* 객체 배열 : 회원 정보를 구조화해 저장
* `sort((a, b) => ...)` : 나이 기준 커스텀 정렬
* `map()`, `join("\n")` : 출력 포맷 구성

---

## 🧩 복잡도

* 시간복잡도 : `O(N log N)` (정렬)
* 공간복잡도 : `O(N)` (회원 객체 배열)
