## 🧩 문제 설명

듣도 못한 사람 목록과 보도 못한 사람 목록이 주어질 때,  
두 목록에 모두 포함된 사람(듣도 보도 못한 사람)을 사전순으로 출력하는 문제이다.

---

## 🧩 문제 핵심 포인트

* 교집합을 빠르게 구하려면 리스트보다 `set`이 유리하다.
* 출력은 사전순(오름차순) 정렬이 필요하다.
* 결과 개수와 이름 목록을 포맷에 맞게 출력해야 한다.

---

## 🧩 내 풀이 방식

* 입력받은 두 문자열 리스트를 각각 `set`으로 변환한다.
* `result = listenArrS & WatchArrS`로 교집합을 구한다.
* 교집합을 리스트로 바꾼 뒤 `sort()`로 오름차순 정렬한다.
* 첫 줄에 개수, 이후 줄에 이름들을 출력한다.

---

## 🧩 코드 로직 정리

### 전체 흐름

```txt
1. n, m 입력
2. 듣도 못한 명단과 보도 못한 명단을 각각 리스트로 입력
3. 두 리스트를 set으로 변환
4. 교집합(result) 계산
5. result를 리스트로 변환 후 오름차순 정렬
6. 개수와 이름 목록을 형식에 맞게 출력
```

---

## 🧩 코드 구현 (내 풀이)

```python
import sys

sys.stdin = open("../예제.txt")
input = sys.stdin.readline

n,m = map(int, input().split())

listenArr = [ input().strip() for _ in range(n)]
WatchArr = [ input().strip() for _ in range(m)]

listenArrS = set(listenArr)
WatchArrS = set(WatchArr)

result = listenArrS & WatchArrS
resultArr = list(result)
resultArr.sort(reverse=False)
Rlength= len(result)
print(str(Rlength)+"\n"+"\n".join(resultArr))
```

---

## 🧩 사용된 Python 개념

* `set()` : 중복 제거와 빠른 탐색/연산을 위한 집합 자료형
* `&` : 두 집합의 교집합 계산
* `list.sort()` : 문자열 리스트 사전순 정렬 (기본 오름차순)
* `"\n".join(list)` : 여러 문자열을 줄바꿈으로 연결해 출력

## ⚠️ 오답 포인트

* 리스트 상태에서 `in` 검사로만 풀면 시간복잡도가 커져 비효율적이다.
* 정렬을 생략하면 출력 순서 조건을 만족하지 못한다.
* `strip()`을 안 쓰면 개행 문자가 포함되어 비교가 틀어질 수 있다.
