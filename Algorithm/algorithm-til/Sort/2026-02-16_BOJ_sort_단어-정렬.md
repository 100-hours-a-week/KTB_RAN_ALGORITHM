## 🧠 문제 키워드

* 사용 알고리즘 : 정렬 + 중복 제거
* 1차 기준: 문자열 길이 오름차순
* 2차 기준: 길이가 같으면 사전순
* 중복 단어는 1개만 유지
* 문자열 비교 함수 사용 (`localeCompare`)

---

## 🧩 문제 설명

N개의 단어가 주어질 때,

1. 길이가 짧은 순
2. 길이가 같으면 사전순
3. 중복 단어 제거

조건으로 정렬해 출력하는 문제다.

---

## 🧩 제한사항

* `1 ≤ N ≤ 20,000`
* 단어 길이: 최대 50
* 알파벳 소문자

## 🧩 문제 핵심 포인트

* 정렬 기준이 2개(길이, 사전순)라 비교 함수를 명확히 작성해야 한다.
* 중복 제거를 먼저 하고 정렬해야 출력 조건을 만족한다.
* 문자열 비교는 `localeCompare`를 쓰면 사전순 비교가 명확하다.

---

## 🧩 내 풀이 방식

* 입력 단어 배열에서 `Set`으로 중복을 제거한다.
* 정렬 시:
  * 길이가 다르면 길이 차이로 비교
  * 길이가 같으면 `localeCompare`로 비교
* 정렬된 결과를 줄바꿈으로 출력한다.

---

## 🧩 코드 로직 정리

### 전체 흐름

```txt
1. 단어 배열 입력
2. Set으로 중복 제거
3. 비교 함수로 정렬
   - 길이 우선
   - 길이 같으면 사전순
4. 결과 출력
```

---

## 🧩 코드 구현 (내 풀이)

```js
const fs = require("fs");
const input = fs.readFileSync(0, "utf-8").trim().split("\n");

function solution(words) {
  const uniqueWords = [...new Set(words)];

  uniqueWords.sort((a, b) => {
    if (a.length === b.length) return a.localeCompare(b);
    return a.length - b.length;
  });

  return uniqueWords;
}

const n = Number(input[0]);
const words = input.slice(1, n + 1);

console.log(solution(words).join("\n"));
```
