## 🧩 문제

* **문제 제목** : 좌표 정렬하기
* **문제 레벨** : Silver V
* **문제 유형** : 정렬 (Sort), 다중 기준 정렬
* **문제 제공** : BOJ
* **문제 링크** : https://www.acmicpc.net/problem/11650
* **코드 파일** : [coordinate-sort-11650.js](../../algorithm-JS/Sort/coordinate-sort-11650.js)

---

## 🧩 문제 설명

2차원 평면 위의 점 `N`개가 주어진다.  
`x`좌표가 증가하는 순으로, `x`가 같으면 `y`좌표가 증가하는 순으로 정렬해 출력하는 문제다.

---

## 🧩 문제 핵심 포인트

* 정렬 기준이 2개다: `x` 오름차순, 동률이면 `y` 오름차순.
* 입력 문자열을 숫자 좌표로 파싱해야 한다.
* 출력은 각 줄에 `"x y"` 형식으로 맞춰야 한다.

---

## 🧩 내 풀이 방식

* 입력 각 줄을 분리해서 `{ x, y }` 객체 배열을 만든다.
* `sort` 비교 함수에서:
  * `a.x === b.x`면 `a.y - b.y`
  * 아니면 `a.x - b.x`
* 정렬된 배열을 `"x y"` 문자열로 변환 후 줄바꿈으로 출력한다.

---

## 🧩 코드 로직 정리

### 전체 흐름

```txt
1. n, 좌표 문자열 배열 입력
2. 각 줄을 x, y 숫자로 파싱해 dots 배열 생성
3. sort로 x 오름차순, x 같으면 y 오름차순 정렬
4. "x y" 형식으로 변환
5. 줄바꿈(join) 후 출력
```

---

## 🧩 코드 구현 (내 풀이)

```js
const fs = require("fs");
const input = fs.readFileSync(0, "utf-8").trim().split("\n");

function solution(n, arr) {
  const dots = [];

  for (const a of arr) {
    const [x, y] = a.split(" ").map(Number);
    dots.push({ x, y });
  }

  dots.sort((a, b) => {
    if (a.x === b.x) return a.y - b.y;
    return a.x - b.x;
  });

  return dots.map((dot) => `${dot.x} ${dot.y}`).join("\n");
}

const n = Number(input[0]);
const arr = input.slice(1, n + 1);

console.log(solution(n, arr));
```

---

## 🧩 사용된 JS 개념

* `split(" ")` + `map(Number)` : 좌표 파싱
* 객체 배열 : `{ x, y }` 구조 저장
* `sort((a, b) => ...)` : 다중 기준 정렬
* `map()`, `join("\n")` : 출력 포맷 변환

---

## 🧩 복잡도

* 시간복잡도 : `O(N log N)` (정렬)
* 공간복잡도 : `O(N)` (좌표 저장 배열)
